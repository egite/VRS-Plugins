Logo Markers

General

About this plugin: Replaces the default aircraft SVG icons on the map with composite images combining the operator's logo and a heading arrow, making operators identifiable at a glance.

Where to get logos: BMPs are read from VRS's Operator flags folder (set in Tools → Options → Data Sources). The community pack at rikgale/VRSOperatorFlags is the usual source — extract its OperatorFlags.zip into that folder. The companion fetch_missing_logos.py script (in the VRS-Plugins repo) can pull just the codes your feed actually sees, using the MissingLogos plugin's log.
