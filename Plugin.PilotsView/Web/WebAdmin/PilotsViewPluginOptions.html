Pilot's View

General

Seconds between camera updates in Google Earth (1-10).

Degrees from vertical. 0 = straight down, 90 = horizon, 80 = pilot's view (default).

About this plugin: Serves a per-aircraft KML file for live tracking inside Google Earth. Desktop clients get a NetworkLink that updates the camera position in real time; mobile clients get a static snapshot to avoid Google Earth's loading issue.
